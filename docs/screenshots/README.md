## How to Add Screenshots

1. Launch the application: `./run.sh` (or `python -m gtk_gui_pro.app`)
2. Navigate to each view
3. Take screenshots using your system's screenshot tool
4. Save them in this directory with the exact filenames listed above
5. Ensure images are in PNG format for best quality

## Image Guidelines

- **Format:** PNG (preferred) or JPG
- **Size:** ~900x600px or similar widescreen ratio
- **Quality:** High resolution, clear text
- **Theme:** Dark theme enabled (professional appearance)
- **Content:** Real or realistic sample data (not empty views)
